#  JuveWorld -- Portfolio Project--

## Code Institute Portfolio Project-01.

![Mockup](https://github.com/NewViewNewBeginning/project-01-CI/blob/main/assets/md-img/amIresposive-mockup.png)

Website presents history of Juventus footbal team and it's best players.
Users will be able to find interesting content in easy and simple way.

This project is for educational/training purposes and can not be used as a template for a business use.
Main aim is to build responsive website with HTML and CSS only and little of JavaScript. I used flexbox & css grid to keep website responsive.
Logo and all branding informations belong to Juventus Club Spaa.

# UX

## Website owner business goals

The main reason to create this website is to keep all information about Juventus team history, players and news in the same place. For all fans of Juventus team and football itself. 

![Welcome Text](https://github.com/NewViewNewBeginning/project-01-CI/blob/main/assets/md-img/welcome-text.png)

## User goals

### New user goals:

- Users are able to find information about the team and players.
- Users can easily fill in forms for all updates, news and add-ons.

### Returning user goals:

- Users can learn more about the team.

### As a business owner:

- I need to make sure that my current and new customers will find a professional help with needs of information.
- I would like to build and maintain relationships with potential and current customers.

### As a new customer:

- I wish to find information about players and team.
- I want to join a fan club to receive all deals and events directly connected with the Juventus team.

### As a returning customer:

- All updates, news and information about team/players/events.

## Structure of the website

The website is designed to be easy and user-friendly on all types of devices. On a desktop, tablet or mobile device there should be no difference for a user to have a fantastic experience. All parts are designed to achieve maximum user satisfaction. Users will get some interaction from the interface as links, buttons and inputs will have a hover effect.

# Deployment

The project was deployed on GitHub Pages. I used VS Code as a development environment where I commited all changes to git version control system.
I used push command in VS Code to save changes into GitHub.

To deploy a project I had to:

- Log in to GitHub and click on repository to deploy ([PP1](https://github.com/NewViewNewBeginning/project-01-CI))
- select `Settings` and find GitHub Pages section at the very bottom of the page
- from source select `none` and then `Milestone-Projects` branch.
- click `save` and page was deployed after auto-refresh.
  > Your site is published at https://newviewnewbeginning.github.io/project-01-CI/

To run localy:

- Log in to GitHub and click on repository to download ([PP1](https://github.com/NewViewNewBeginning/project-01-CI))
- select `Code` and click Download the ZIP file.
- after download you can extract the file and use it in your local environment

Alternatively you can [Clone](https://docs.github.com/en/free-pro-team@latest/github/creating-cloning-and-archiving-repositories/cloning-a-repository)
or [Fork](https://docs.github.com/en/free-pro-team@latest/github/getting-started-with-github/fork-a-repo)
this repository ([PP1](https://github.com/NewViewNewBeginning/project-01-CI)) into your github account.
